📖 프로젝트 소개

BookStamp는 독서 기록을 아름답고 직관적으로 관리할 수 있는 모바일 애플리케이션입니다. 고풍스러운 책장 디자인과 함께 당신의 독서 여정을 기록하고, 읽은 책들을 시각적으로 아름답게 보관할 수 있습니다.

🎯 다양한 API를 활용한 책 검색 서비스 개발 중

✨ 주요 기능

🔍 스마트 검색

알라딘 API와 Google Books API를 활용한 빠르고 정확한 책 검색

📚 비주얼 서재

고풍스러운 책장에 책을 배치하는 아름다운 UI/UX

📝 독서 기록

책에 대한 메모와 독서 횟수를 기록하고 관리

🎨 표지 자동 수집

책 검색 시 표지 이미지 자동 다운로드 및 저장

🛠 기술 스택

React Native Expo TypeScript SQLite 알라딘 OpenAPI Google Books API

🎯 개발 목적

독서를 좋아하는 사람들이 자신의 독서 기록을 아름답고 의미있게 보관할 수 있도록 돕고자 합니다. 단순한 리스트가 아닌, 시각적으로 만족스러운 나만의 서재를 디지털 공간에 구현하는 것이 목표입니다.

📖 About the Project

BookStamp is a mobile application that allows you to manage your reading records beautifully and intuitively. With its classic bookshelf design, you can document your reading journey and visually preserve your read books in an elegant way.

🎯 Developing Book Search Service with Multiple APIs

✨ Key Features

🔍 Smart Search

Fast and accurate book search using Aladin API and Google Books API

📚 Visual Library

Beautiful UI/UX featuring books arranged on a classic bookshelf

📝 Reading Records

Record and manage notes and reading counts for each book

🎨 Auto Cover Collection

Automatic download and storage of book cover images when searching

🛠 Tech Stack

React Native Expo TypeScript SQLite Aladin OpenAPI Google Books API

🎯 Development Purpose

We aim to help book lovers preserve their reading records in a beautiful and meaningful way. Our goal is to create a visually satisfying personal library in the digital space, rather than just a simple list.

📖 Über das Projekt

BookStamp ist eine mobile Anwendung, mit der Sie Ihre Leseaufzeichnungen schön und intuitiv verwalten können. Mit dem klassischen Bücherregal-Design können Sie Ihre Lesereise dokumentieren und gelesene Bücher elegant visuell aufbewahren.

🎯 Entwicklung eines Buchsuchdienstes mit mehreren APIs

✨ Hauptfunktionen

🔍 Intelligente Suche

Schnelle und genaue Buchsuche mit Aladin API und Google Books API

📚 Visuelle Bibliothek

Schöne UI/UX mit Büchern auf einem klassischen Bücherregal

📝 Leseaufzeichnungen

Notizen und Lesezählungen für jedes Buch aufzeichnen und verwalten

🎨 Auto-Cover-Sammlung

Automatisches Herunterladen und Speichern von Buchcover-Bildern bei der Suche

🛠 Tech Stack

React Native Expo TypeScript SQLite Aladin OpenAPI Google Books API

🎯 Entwicklungszweck

Wir möchten Buchliebhabern helfen, ihre Leseaufzeichnungen auf schöne und bedeutungsvolle Weise zu bewahren. Unser Ziel ist es, eine visuell zufriedenstellende persönliche Bibliothek im digitalen Raum zu schaffen, statt nur einer einfachen Liste.
