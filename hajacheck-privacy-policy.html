개인정보처리방침

최종 업데이트: 2026년 4월 8일

핵심 요약: 당신의 데이터는 기기 안에만 있습니다

하자체크는 서버가 없는 완전 로컬 앱입니다. 촬영한 사진, 점검 데이터, PDF 보고서 등 모든 정보는 오직 사용자의 기기에만 저장됩니다. 계정 가입이 필요 없으며, 개인정보를 수집·저장·판매하지 않습니다.

1. 소개

본 개인정보처리방침은 하자체크("앱", "당사")가 어떤 정보를 수집하고 사용하는지 설명합니다. 하자체크는 현장 하자점검 및 PDF 보고서 생성을 위한 도구입니다. 당사는 사용자의 프라이버시를 보호하고 투명하게 운영할 것을 약속합니다.

2. 수집하는 정보

2.1 앱이 수집하지 않는 정보

하자체크는 다음 정보를 수집하지 않습니다:

  • 이름, 이메일 주소, 전화번호 등 개인 식별 정보
  • 위치 정보 (GPS)
  • 연락처, 캘린더, 브라우징 기록
  • 광고 식별자
  • 사용 패턴 분석 데이터
  • 기기의 다른 앱이나 파일 접근

2.2 기기에만 저장되는 정보

다음 정보는 사용자의 기기 내부 저장공간에만 저장되며, 외부 서버로 전송되지 않습니다:

  • 점검 프로젝트: 프로젝트명, 주소, 날짜 등 사용자가 직접 입력한 정보
  • 하자 항목: 카테고리, 심각도, 메모, 담당자, 마감일 등
  • 사진: 카메라로 촬영하거나 갤러리에서 가져온 현장 사진
  • 업체 정보: 설정에서 입력한 회사명, 전화번호, 주소
  • 앱 설정: 언어 설정, 결제 상태

2.3 RevenueCat (인앱 결제)

하자체크 Pro 구매 처리를 위해 RevenueCat을 사용합니다. RevenueCat은 다음 정보를 처리합니다:

  • 익명 사용자 식별자 (앱이 자동 생성, 개인 식별 불가)
  • 구매 상태 및 영수증 정보

결제 정보 및 Apple ID는 Apple이 직접 처리하며, 당사는 접근할 수 없습니다.

3. 정보 사용 방법

  • 점검 데이터 저장: 사용자가 입력한 하자 항목과 사진을 기기에 저장
  • PDF 보고서 생성: 저장된 데이터를 기반으로 PDF 보고서를 기기 내에서 생성
  • 결제 상태 확인: RevenueCat을 통해 Pro 구매 여부 확인

4. 데이터 저장 및 보안

4.1 로컬 저장

모든 데이터는 사용자의 기기 내 SQLite 데이터베이스와 파일 시스템에만 저장됩니다. iOS의 앱 샌드박스 보안 정책에 의해 다른 앱은 이 데이터에 접근할 수 없습니다.

4.2 외부 서버 없음

하자체크는 자체 서버를 운영하지 않습니다. 앱이 생성하는 모든 데이터는 인터넷을 통해 전송되지 않습니다.

주의: 앱 삭제 시 데이터 복구 불가

앱을 삭제하면 모든 점검 데이터와 사진이 영구적으로 삭제됩니다. 당사는 데이터를 별도로 보관하지 않으므로 복구가 불가능합니다. PDF로 내보내거나 사진을 갤러리에 저장하는 방식으로 중요한 데이터를 백업하시기 바랍니다.

5. 제3자 서비스

서비스목적공유 데이터
RevenueCat인앱 결제 관리익명 사용자 ID, 구매 상태
Apple App Store결제 처리당사 미관여 (Apple이 직접 처리)

당사는 광고 네트워크, 행동 추적, 분석 서비스를 사용하지 않습니다.

6. 앱 권한

권한목적필수 여부
카메라현장 하자 사진 촬영핵심 기능
사진 라이브러리기존 사진을 하자 항목에 첨부선택

하자체크는 위치 정보, 마이크, 연락처, 캘린더에 접근하지 않습니다.

7. 인앱 결제

  • 결제는 Apple이 처리하며 RevenueCat으로 관리됩니다
  • 당사는 결제 정보나 Apple ID를 수신하지 않습니다
  • RevenueCat은 익명 사용자 식별자와 구매 상태만 수신합니다
  • 동일한 Apple ID로 모든 기기에서 구매 복원이 가능합니다

8. 데이터 보존

  • 점검 데이터 및 사진: 앱 삭제 또는 사용자가 직접 삭제할 때까지 기기에 보관
  • 결제 상태: RevenueCat 서버에 구매 이력 보관 (Apple 정책에 따름)

앱을 삭제하면 기기의 모든 로컬 데이터가 제거됩니다.

9. 아동 개인정보 보호

하자체크는 만 14세 미만 아동을 대상으로 하지 않습니다. 앱은 계정 생성이 필요 없으며 개인정보를 수집하지 않습니다.

10. 사용자 권리

  • 앱 내에서 모든 점검 데이터 확인 및 삭제 가능
  • 앱 삭제 시 모든 로컬 데이터 영구 삭제
  • iOS 설정 > 하자체크에서 카메라 및 사진 접근 권한 변경 가능

11. 본 방침의 변경

본 개인정보처리방침은 변경될 수 있습니다. 변경 시 페이지 상단의 최종 업데이트 날짜를 갱신하고 해당 URL에 새 방침을 게시합니다. 변경 후 앱을 계속 사용하면 변경된 방침에 동의한 것으로 간주됩니다.
