2. Description of Service

VaultX is a local password manager for securely storing passwords, account credentials, and sensitive information. Key features include:

  • AES-256 encrypted local password storage
  • Biometric unlock via Face ID and Touch ID
  • iOS AutoFill integration for apps and browsers
  • Password health check and breach detection
  • Decoy Mode for coercion scenarios (Premium)
  • SOS Emergency Recovery via secret sharing (Premium)
  • Optional iCloud sync across devices (Premium)

3.2 Local Data Storage

All passwords and credentials are encrypted with AES-256 and stored exclusively on your device. They are never transmitted to our servers.

Important: Master Password & Data Recovery

VaultX is designed with zero-knowledge architecture. Please be aware of the following:

  • Your master password is never stored on any server — if you forget it, your vault cannot be recovered by us
  • Deleting the app permanently removes all locally stored passwords and data
  • Device loss or reset will result in unrecoverable data loss unless iCloud sync is enabled
  • We strongly recommend using the SOS Emergency Recovery feature to safeguard your master password
  • In-app purchases (Premium) can be restored using the same Apple ID
